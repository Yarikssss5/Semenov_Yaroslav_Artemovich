# Tauri + Yew

This template should help get you started developing with Tauri and Yew.



Please Download and place Tailwindcss CDN Version to public folder by this (link)[https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4] and save this page as tailwindcss.js file


Requirements :


Rust dependencies:
```shell
cargo install tauri-cli
cargo install trunk
cargo install wasmbindgen-cli
rustup target add wasm32-unknwown-unknwon
```


Run : 

```shell
docker compose up -d
cargo tauri dev
```
